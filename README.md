## 🛠️ Fixing common install failures

Some users encounter errors during the installation process on Windows 11. These errors often occur due to security settings or conflicts with other monitoring tools.

### Check your anti-virus settings
Sometimes, security software blocks the installation of hardware monitoring tools because these programs access low-level system sensors. If the installer stops or shows an error, temporarily disable your anti-virus software. Try to run the installer again. Turn your security features back on once the installation finishes.

### Update your hardware drivers
Outdated drivers can prevent the software from reading your sensors correctly. Visit your motherboard manufacturer website to download the latest chipset and sensor drivers. Restart your computer after updating your drivers and attempt the software installation again.

### Run in compatibility mode
Windows 11 may treat older installers as incompatible. If you see an error message, follow these steps:

1. Right-click the installation file.
2. Select Properties.
3. Click the Compatibility tab.
4. Check the box labeled "Run this program in compatibility mode for."
5. Choose "Windows 10" from the drop-down menu.
6. Click Apply and then OK.
7. Attempt to run the installer again.

### Clear temporary files
Old files from failed previous installations can interfere with new setup attempts. Navigate to your Temp folder by typing `%temp%` into the Windows search bar. Delete the contents of this folder. Restart your system and try the installation one more time.

## 🖥️ How to use the software

Once installed, double-click the HWMonitor Pro icon on your desktop. The main window displays a list of your computer hardware. 

The software categorizes hardware components by name. You can expand each section by clicking the plus sign (+) next to the component. 

* **Temperatures**: View heat levels for your processor and graphics card. High temperatures indicate a need for better case ventilation or dust cleaning.
* **Voltages**: Ensure your power supply provides steady energy to your components.
* **Fans**: Monitor fan speeds in rotations per minute. This helps you determine if your cooling system functions correctly under load.

## 💡 Important usage tips

- **Check values under load**: Run a demanding game or a heavy application to see how your computer performs under stress.
- **Identify overheating**: Most processors operate safely below 80 degrees Celsius. If you see values consistently higher than this, check your fans for dust.
- **Keep it simple**: If you do not recognize a system sensor, avoid changing settings. The default display provides enough data for most users.
